Admin Dashboard – Frontend Project

A fully responsive Admin Dashboard built from scratch using HTML5, CSS3, JavaScript, and Bootstrap 5. Designed for monitoring projects, managing tickets, tracking tasks, and visualizing data with interactive charts. 
Supports light and dark mode for enhanced user experience.

Features
* Light and Dark Mode toggle for user preference 
* Messages Section – quick view of recent messages or notifications 
* To-Do List Section – manage tasks with add, markthrough, and remove functionality 
* Charts Section – visualize data with:
    * Bar chart 
    * Pie chart 
    * Histogram chart 
* Ticketing Section – track support tickets or tasks 
* Project Status Section – overview of ongoing projects and progress 
* Calendar Section – view events, deadlines, and schedules 

Technologies Used
* HTML5 
* CSS3 
* Bootstrap 5 
* JavaScript (Vanilla JS) 
* Chart.js
